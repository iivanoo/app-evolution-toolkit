Welcome to the Airpush SDK for Android!

Included in this distribution is everything you need to integrate Airpush ads into your Android applications. Get started by reading the Standard SDK installation guide at http://manage.airpush.com/docs/index.php?title=Standard_SDK_1.0

You can download the demo project "Standard demo" using the following link http://manage.airpush.com/sdk/StandardDemo.zip. This demo provides an overview on how to integrate the Airpush Standard SDK into your app. If you have any issues integrating the SDK, please feel free to email publishersupport@airpush.com and our support team will get them resolved ASAP.
